Features
Interactive Text-Based Interface: Clean and intuitive command-line interface (CLI) for executing drawing commands.
2D Shape Rendering: Supports creating and manipulating fundamental geometric shapes (e.g., points, lines, rectangles, and circles).
Canvas Manipulation: Dynamic grid/canvas management allowing real-time rendering of shapes using text characters.
Layer/Object Management: Ability to add, select, modify, or delete shapes from the active canvas project.
Robust Input Handling: Built-in validation to handle user inputs gracefully and prevent program crashes during rendering.

Technologies Used
Language: C++ (C++11 or higher)
Paradigm: Object-Oriented Programming (OOP) for modular shape and canvas management.
Standard Library: <iostream>, <vector>, <string>, and <memory> for efficient memory and data management.
Development Environment: Visual Studio Code / Terminal.
